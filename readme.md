Buddha's Birthday Greeting Card
This project is a web-based greeting card celebrating Buddha's Birthday, also known as Vesak or Buddha Purnima. It includes a beautiful image of Buddha, inspirational quotes, and interactive features such as changing the font style, background color, and displaying more information about the holiday.

Features
Inspirational Quotes: Display random quotes from Buddha.
Change Font Style: Change the font style of the quotes.
Change Background Color: Change the background color of the page.
More Info Modal: Display a modal with more information about Buddha's Birthday.

Getting Started
To get a local copy up and running, follow these simple steps.

Prerequisites
Make sure you have a modern web browser installed.

Installation
Download the project files.
Open the index.html file in your web browser to view the project.
Usage
Click on the Change Font Style button to cycle through different font styles for the quote.
Click on the Change Background Color button to change the background color randomly.
Click on the New Quote button to display a new quote.
Click on the More Info button to open a modal with more information about Buddha's Birthday.
Click on the close button (×) or anywhere outside the modal to close the modal.
